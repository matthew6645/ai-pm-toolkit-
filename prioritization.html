How this works
Claude applies RICE (Reach, Impact, Confidence, Effort) scoring to each feature in the context of the stated strategic goal. Rather than asking Claude to "rank these features," the prompt instructs it to be opinionated and explain trade-offs — because a prioritization without reasoning is just a list. The strategic goal grounds the scoring, so the same features would rank differently depending on what you're trying to achieve.
